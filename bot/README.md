# Discord bot

Stub for the NMS10 Discord bot. Final tech choice TBD — Python (discord.py) or Node (discord.js v14, matches viobot stack).

## Slash commands (planned)

- `/submit-base` — modal for base details, posts to `POST /api/bases`
- `/submit-meetup` — modal for meetup details, posts to `POST /api/meetups`

## Notification channels (planned)

- New base submitted (admin channel)
- New base approved (public channel)
- New social post detected (public channel)
- New meetup added (public channel)

Real implementation lands in Phase 1.5.
